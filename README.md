# Primer theme for Obsidian

WIP WIP WIP

## Development

Setup & Build:

```bash
yarn

yarn watch # or yarn build for production
```

Then link the theme styles to obsidian vault directory:

```bash
ln -s ./dist/theme.css ./vault/.obsidian/theme.css
```

And load it with CSS Snippet.
